What I Did with the Data

üìä Data Prep

  • Started with 15 numeric features (ages, amounts, that kind of thing)
  • Plus 11 categorical ones (states, occupations, etc.)
  • Broke down dates into useful parts (day, month, which day of week)
  • Used frequency encoding when there were too many categories
  • One-hot encoded the simpler categorical stuff
  • Filled in missing values and scaled everything properly

ü§ñ The Baseline Model

  • Went with Logistic Regression to start
  • Used 'balanced' class weights to handle the imbalance
  • Solver: liblinear (works well for this)
  • 1000 max iterations
  • 80-20 train-test split

üí° This is just the baseline - Random Forest is next on the list for experimentation!

Since this is the baseline model and there's more to come yet, I'll be adding Random Forest to try experimentation around non-linear patterns and feature interactions. Keeping it simple for now to see what works.
